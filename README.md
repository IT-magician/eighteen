## 🎯 MSA 설계

### 배포

서비스별 배포가 가능.(배포시 전체 서비스의 중단이 없음)
특정 서비스의 요구사항만 반영하여, 빠르게 배포 가능.
확장

다양하고, 새로운 서비스에 대한 유연한 확장 가능.
장애

일부 서비스의 장애가 전체 서비스로 확장될 가능성을 차단. (User service 에 해당하는 장애가 Owner service에 영향을 미치지 ❌)
부분적 장애에 대한 격리 처리 가능.
